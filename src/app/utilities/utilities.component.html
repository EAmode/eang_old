Utilities

Sizing

The following attributes are available to allow you to re-size a given component. Components in eang are largely based off em unit values which means that their values are proportional to the the parent font. These attributes adjust the parent font of the element to adjust all values in the component acoordingly.

  • [sm]
  • [md]
  • [lg]
  • [xl]
  • [xxl]

Flex Box

Flexbox is a powerful tool for 2-D layouts. We capture different common use-cases for flex box in the attributes below. The goal is to reduce the redundancy of common flexbox layouts, e.g. vertical and horizontal centering, and spaceing children elements out evenly.

[flex-center]

<div [flex-center]>

[flex-space-evenly]

<div [flex-center]>

CSS Grid

CSS Grid is another powerful layout tool. We use common, and advanced grid patterns to simplify some of the capabilities of Grid, e.g. responsive multi-column layouts.

[ multi-col ]

<div [center]>
<div [center]>

[multi-col] [center]

<div [center]>
<div [center]>

Margins & Padding

Eang offers configurable margin and padding settings as well as attributes to implement them in your markup. Global CSS Custom Properties set these values which allow developers to set these values at the theme level and the abillity to override them for more specific component scopes if desired.
