Getting Started

It's easy to get started using Eang in your next project. Prerequisite: Install Angular 7.

Create a new project use Angular CLI:
Add `eang` to your Angular project by running:
Import `eang` modules in your project module file. e.g. app.module.ts :
To use eang themes add @import '~@eamode/eang/css/all.css' to yourstyles.css file.
In index.html add a class to the body tag to apply the selected theme.
Theme Name Description
mode default theme of MODE
material Components designed following Google's Material design
your-theme-here It is easy to add your own completely customized theme as well.
Launch your app by running npm start. Enjoy.
