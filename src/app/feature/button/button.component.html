Button

Buttons allow users to take actions, and make choices, with a single tap. Eang has 4 core types of buttons. Add ea-button class in <button> tag to apply styles.

You can do the same with the <div>, <span>, <a> tags, but need also to add role="ea-button".

Icon Buttons

Eang makes it easy to add icons into your buttons with a preset lay out that also scales up proportionally based off font-size. the [icon] attribute can be added to a button to take care of some basic icon work.
