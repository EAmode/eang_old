Inputs

The <input> tag allows you to create different interface elements and provide user interaction. Mainly <input> is intended for creating text fields, various buttons, radio buttons and checkboxes. The main attribute of the <input> tag is "type" which defines the type of element such as text field, field to insert the password, date time picker etc. To styling the inputs add ea-form-field class for a wrapper.
